String e um tipo de dado primitivo em Python

string são textos que estão dentro de aspas

\ --> caractere de escape usado antes de um comando para ignorá-lo dentro de um string
  print('Esse é meu \'texto\' (str)') --> usando \ para ignorar as aspas

\n --> usado para quebrar uma linha
  print('um texto \n quebrado')
